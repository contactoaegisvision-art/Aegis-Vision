# Aegis Vision Tech S.A.S. - Landing Page

Bienvenido al repositorio de la página web oficial de **Aegis Vision Tech S.A.S.**, un emprendimiento nacido de la convergencia tecnológica entre la seguridad física y la transformación digital.

## Propuesta de Valor
Seguridad híbrida que combina Inteligencia Artificial, Drones y Personal Humano para ofrecer:
* Monitoreo 24/7 sin puntos ciegos.
* Analítica predictiva de delitos.
* Reducción de costos operativos.

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica del sitio web.
* **CSS3:** Estilos responsivos utilizando variables para la paleta corporativa (Azul Medianoche y Naranja Neón).
* **FontAwesome:** Implementación de íconos vectoriales.

## Catálogo de Productos
1. Kit de Cámaras con Reconocimiento Facial.
2. Patrullaje perimetral con Drones "Aegis-Fly".
3. Sistema de Alarma Inteligente con enlace a App.

## Cómo ejecutar el proyecto
1. Clona este repositorio o descarga el código.
2. Asegúrate de tener la imagen del logotipo guardada como `logo.png` dentro de una carpeta llamada `assets/`.
3. Abre el archivo `index.html` en cualquier navegador web moderno.
